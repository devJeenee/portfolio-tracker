# 포트폴리오 종합 분석

프로젝트의 포트폴리오 가치를 종합 분석합니다.

## 실행할 작업

`portfolio_insight` MCP 도구를 사용하여 다음을 분석해주세요:

1. **강점 분석** (strengths): 현재 프로젝트에서 잘 구현된 패턴과 기술
2. **경험 발굴** (experiences): Git 히스토리에서 포트폴리오에 쓸 수 있는 개선 경험 추출
3. **개선 기회** (opportunities): 포트폴리오 가치가 높은 개선 포인트
4. **성숙도 점수** (maturity): 0-100점 프로젝트 성숙도 + 등급(S~F)

## 파라미터

- `path`: $ARGUMENTS (비어있으면 현재 디렉토리)
- `sections`: ["all"]

## 출력 형식

결과를 한국어로 보기 좋게 정리해주세요:
- 강점은 카테고리별로 그룹핑
- 경험은 impact 순서대로
- 개선 기회는 portfolioValue 순서대로 상위 10개
- 성숙도는 점수 + 등급 + 카테고리별 점수 + 개선 로드맵
