# 개선 기회 탐색

현재 코드에서 포트폴리오 가치가 높은 개선 포인트를 찾습니다.

## 실행할 작업

`portfolio_insight` MCP 도구를 사용하세요:

- `path`: $ARGUMENTS (비어있으면 현재 디렉토리)
- `sections`: ["opportunities"]

## 출력 형식

개선 기회를 portfolioValue 순서로 정리하고, 각 항목에 대해:
- 타입 + 심각도 + 포트폴리오 가치 점수
- 파일 위치 + 문제 설명
- **구체적인 개선 방법** (어떻게 고치면 되는지)
- 개선 후 포트폴리오에서 어필할 수 있는 포인트

를 한국어로 설명해주세요. 상위 15개만 보여주세요.
