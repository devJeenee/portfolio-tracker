# 프로젝트 성숙도 평가

프로젝트의 전반적인 성숙도를 0-100점으로 평가합니다.

## 실행할 작업

`portfolio_insight` MCP 도구를 사용하세요:

- `path`: $ARGUMENTS (비어있으면 현재 디렉토리)
- `sections`: ["maturity"]

## 출력 형식

1. **종합 점수**: 0-100점 + 등급(S/A/B/C/D/F)
2. **카테고리별 점수**: 각 카테고리(아키텍처, 보안, 테스트 등)의 점수와 감점 사유
3. **개선 로드맵**: 우선순위별 개선 항목 (포트폴리오 가치 높은 순서)

를 한국어로 보기 좋게 정리해주세요.
